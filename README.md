## 🧠 How It Works

```
Input Image
     │
     ├──► EfficientNetV2B2 (weight: 43%) ──► P(real)
     │
     └──► ResNet101V2      (weight: 57%) ──► P(real)
                                                │
                              Weighted Average (0.43 + 0.57)
                                                │
                                      Final Prediction
                                   REAL / FAKE + Risk Level
```

Two pretrained models independently score each image. Their predictions are combined using weights found by grid search on the validation set. The ensemble reduces individual model errors — when one misclassifies, the other often corrects it.

## 🏗️ Architecture

### Models
- **EfficientNetV2B2** — pretrained on ImageNet, detects GAN-generated texture artifacts using compound scaling (width + depth + resolution scaled together)
- **ResNet101V2** — pretrained on ImageNet, captures structural facial inconsistencies using 101-layer deep residual connections

### Classification Head (same for both models)
```
GlobalAveragePooling2D
        ↓
   Dropout(0.5)
        ↓
   Dense(256, ReLU)
        ↓
   Dense(128, ReLU)
        ↓
  Dense(1, Sigmoid)  →  probability of REAL
```

### Training Strategy — Two-Phase Transfer Learning
| Phase | Layers Trained | Learning Rate | Purpose |
|---|---|---|---|
| Phase 1 | Head only (base frozen) | 0.001 | Train classification head |
| Phase 2 | Full model (fine-tuning) | 0.00001 | Adapt pretrained features |

### Ensemble Weight Optimization
Grid search over 101 combinations (0/100 → 100/0 in 1% steps) on validation set.
Best: **EfficientNet 43% + ResNet 57%**

## 🔍 Risk Levels

| Risk | Condition | Meaning |
|---|---|---|
| 🟢 LOW RISK | REAL, confidence ≥ 90% | Highly likely authentic |
| 🟡 MEDIUM RISK | REAL, confidence 75–90% | Probably real, worth reviewing |
| 🟠 HIGH RISK | FAKE, confidence < 80% | Possible deepfake |
| 🔴 CRITICAL | FAKE, confidence ≥ 80% | Strong deepfake signal |

## 🗂️ Dataset

**CIFAKE** — Real vs AI-Generated Images
- Real images from CIFAR-10 categories
- Fake images generated by Stable Diffusion
- Balanced 50/50 split
- Train/Val split: 85% / 15% (stratified)
